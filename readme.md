# InnoEngine

**A simple 3D game engine using C++20 and Vulkan SDL3.**

- [O] - invert y, to usual style
- [O] -  textures render
- [] - shadows
- [] - player actor
- [] - .cae animations
- [] - ragdoll
- [] - car
- [] - terrain LODs
- [] - terrain edit tool 
- [] - connect SDL3
- [] - engine UI (imgui like), but with editor
- [] - game separated
- [] - SDL3 ttf
- - [] - SDL3 mixer
  - - [] - SDL3 net
  -[]  - vma - allocation
- [] - build for arch

- [x] PBR materials
- [x] HDR lighting
- [x] GLTF mesh importer
- [x] Skinned meshes and animations
- [x] Fonts and text
- [x] Dynamic rendering (Vulkan)
- [x] In-game UI

- [x] Shadow mapping
- [x] Audio

## Requirements

### Runtime

  - Windows 10
  - Linux: X, Wayland (untested)
  - Mac OSX: untested, will require MoltenVk
- GPU supporting Vulkan 1.3+, its driver, and loader

### Development

- CMake 3.23+
